## 有道词典数据抓取助手
针对网易有道词典的数据抓取助手，可抓取对应的英文或者中文单词，短语以及例句。

---

## 如何开始
- 首先确保计算机上已经安装有node和npm。windows计算机还应该确保允许脚本执行策略，如果发现运行npm失败，请尝试使用管理员身份打开powershell，并执行：
```powershell
Set-ExecutionPolicy RemoteSigned
```
- 使用vscode打开本目录(注意，是本目录，不是仓库的根目录)，或者直接从终端进入本目录。
- 首先运行npm安装对应依赖：
```powershell
npm install
```
- 接着运行命令即可启动程序：
```powershell
npm start
```
- 本程序中的查询仅仅给出一个测试用例，如果需要从文件读取并自动化抓取数据等功能，还需要针对源码自行修改。本程序暂时仅仅提供了js逆向和数据提取等源码。
